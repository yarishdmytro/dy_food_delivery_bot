Інструкція з розгортання системи на локальному сервері:
  1. На власному локальному сервері потрібно встановити:
  ⎯ Node js(https://nodejs.org/uk/download/);
  2. В директорії проекту потрібно запустити команду для npm install для встановлення залежностей для даного проекту.
  3. Створіть обліковий запис бота
    - Щоб отримати токен авторизації слідуйте інструкціям за посиланням (https://core.telegram.org/bots/#botfather)
  4. Відредагуйте config.js
    - Встановіть YOUR_TELEGRAM_BOT_TOKEN за допомогою токена авторизації вашого бота отриманого у бота @botfather у застостосунку Telegram
    - Встановіть MONGODB_URL за допомогою своєї адреси MongoDB
  5. Для запуску бота потрібно запустити команду:
    - npm run start
    - Щоб зупинити роботу бота натисніть CTRL+C

  6. В застосунку Telegram в пошуку знайдіть бота за ім'ям @dy_food_delivery_bot
